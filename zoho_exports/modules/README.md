# zoho_exports/modules/

This folder contains CRM module and field metadata.

## File Naming Convention

| Pattern | Description |
|---|---|
| `*_summary.json` | Hand-curated, AI-friendly summaries. These are the primary files for AI consumption. DO NOT overwrite manually. |
| `*_raw.json` | Auto-generated by `scripts/pull_zoho.py` from the live Zoho CRM API. Written on every sync. |
| `layouts_*_raw.json` | Auto-generated layout metadata per module. |

## Curated Summary Files (authoritative for AI)

- `modules_summary.json` — all 50 modules with IDs, labels, types
- `fields_Accounts_summary.json` — Employers module fields
- `fields_Contacts_summary.json` — Agencies module fields
- `fields_Deals_summary.json` — Service Profiles module fields
- `fields_Notes_summary.json` — Notes module fields
- `fields_Phone_Calls_summary.json` — Phone Calls (CustomModule5) — includes Retell/Pipedream/Twilio integration tags
- `fields_Providers_summary.json` — Providers (CustomModule1) fields
- `fields_Service_Schedules_summary.json` — Service Schedules (CustomModule2) fields
- `fields_Students_summary.json` — Students (CustomModule7) fields
