# zoho_exports/workflows/

This folder is **auto-populated** by the daily GitHub Action.

## Files written here

| File | Contents |
|---|---|
| `all_workflow_rules.json` | Summary list of all workflow rules (id, name, module, status, trigger) |
| `workflow_rule_details.json` | Full detail of each rule including conditions, actions, and notifications |

## How to use with AI

Ask: *"What workflow rules run on the Deals module? What actions do they perform?"*
