# zoho_exports/users/

This folder is **auto-populated** by the daily GitHub Action.

## Files written here

| File | Contents |
|---|---|
| `all_users.json` | User list with non-sensitive fields: id, full_name, email, role, profile, status, time_zone, locale |
| `org_info.json` | Organisation-level metadata from `/org` |

## Privacy note

The pull script strips sensitive user fields before committing. Only structural/identity fields are stored.

## How to use with AI

Ask: *"How many users are there? What roles/profiles are in use?"*
